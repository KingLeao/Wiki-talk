# wiki-talk-parser

Parse wikipedia dump files (xml) to wiki-talk networks.

## Installation

Download the stand-alone jar file.

## Usage

    $ java -jar wiki-talk-parser-0.1.0-standalone.jar *input-file* > *output-file*

## Compilation

    $ lein uberjar
